Header and Nav Layouts

<header>: CSS Grid

This header is an example of a CSS Grid layout. It uses the display: grid; property to create a grid structure.

The header has three children elements: <img>, <div>, and <nav>

These elements are grid items since their parent is a grid container and so these elements participate in the grid layout.

The header uses the following CSS declarations to create the grid layout: 

display: grid;
 grid-template-columns: 120px 1fr 1fr;
 align-items: center;

As a result, the header is divided into three areas: a logo area, a title area, and a navigation area that are side by side.

  • display makes the header a grid
  • grid-template-columns defines that there are three columns, the first one at a fixed size of 120px and the rest sharing equally the rest of the space
  • align-items aligns the grid items to the middle of the header (the grid)

<nav>: CSS Flexbox

This navigation is an example of a CSS Flexbox layout. It uses the display: flex; property to create a flexible layout.

The nav has one child element: <ul> which has to be the Flex container because its child elements (the list items) need to work together as flex items.

The ul with the class of navigation uses the following CSS declarations to create the flex layout: 

display: flex;
 list-style: none;

As a result, the nav items are evenly distributed within the nav area.

  • display makes the nav a flex container
  • list-style removes the default list styling

The li (list items) flex property is used on the list items to control their size and how they grow or shrink within the nav.

flex: 1 1 auto;
